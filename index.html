Table Properties

Name Formatted Tag Description
Table <table></table> Creates a table
Table Row <tr></tr> Adds row to table
Table Data <td></td> Adds data to a row per colomn
Table head <thead></thead> Creates the head of table
Table heading <th></th> Clarifies a heading
Table body <tbody></tbody> Creates the table body which contains the body of the contents
Table footer <tfoot></tfoot> Creates the table footer

Table Attributes

Name Attribute Description
Column Span <td colspan="x"></td> Allows an element to span a colomn - variable is an integer
Row Span <td rowspan="x"></td> Allows an element to span a row - variable is an integer
